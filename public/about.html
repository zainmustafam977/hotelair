About Us

HotelAir - Hotel Management System

HotelAir is a modern, full-featured hotel management system designed for seamless operations, guest satisfaction, and efficient administration. This project is developed as part of the Web Technologies course at the University of Management and Technology (UMT).

Project & Academic Information
  • University: University of Management and Technology (UMT)
  • Course: IT310 - Web Technologies
  • Section: V4
  • Professor: Muhammad Jawad Farooq
  • Semester: Spring 2024

Project Overview

HotelAir provides a comprehensive platform for hotel management, including:

  • Room and booking management
  • Guest and staff directory
  • Payments and analytics
  • Calendar and dashboard views
  • Notifications and audit logs
  • Modern responsive UI with dark mode
  • Secure authentication and user roles
Technologies Used
  • Node.js, Express.js (Backend API)
  • Microsoft SQL Server (Database)
  • HTML5, CSS3, Bootstrap 5, JavaScript (Frontend)
  • Chart.js, FullCalendar, and more
